Case manager integration with Forms

A concept demo which shows an example of how a case manager could integrate with the GOV.UK Forms product

This prototype has not been tested with users and is only a concept idea

Start now

Features

Features that are currently prototyped are:

  • Filtering and searching for a case
  • Pagination through a list of cases
  • View a case detail
  • View if a payment has been made for a case
  • View attachments that have been made through GOV.UK Forms
  • Change a case status or add a note to a case
  • See activity log of a case

Features that could be prototyped are:

  • Sort by (case status, updated last)
  • Reply to the end-user who made a form submission
  • Analytics of processing cases
  • Account management
  • Third-party integrations
  • Billing
  • Data storage
  • Reminders for appointments and tasks
  • Scheduling and mangaing appointments
  • Automation of tasks
  • Knowledge articles
  • Print a form or case
  • Time recording a case
  • Locking a case after inactivity for a set period of time
